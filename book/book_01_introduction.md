# Introduction
## What this book is about

This book is a practical resource for learning how to master Generative Artificial Intelligence (GenAI) from the ground up.
The approach is hands-on and interactive.
Being about programming, each chapter is designed to be self-contained, with all relevant code and assets available for you to experiment with.

It covers fundamental concepts, practical applications, and hands-on projects to help you become proficient in developing and deploying GenAI solutions.

## Who this book is for
Practitioners.

## Who this book is not for
Novice programmers.
Some experience with programming is required, i.e. post-basic knowledge of Python.
This includes familiarity with Python syntax, data structures, and basic programming concepts, which will not be covered in this repository.
There are many excellent resources for learning Python online, which can be used to supplement this book.
Also consider doing the certifications offered by the Python Institute.
